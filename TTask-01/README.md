# Task-01: Temperature Conversion Program

## Description
This program converts temperature between Celsius, Fahrenheit, and Kelvin based on user input.

## Features
- Accepts temperature value
- Accepts unit (C, F, K)
- Converts into remaining two units
- Displays output clearly

## Technologies Used
- Python

## How to Run
1. Open the file in Python
2. Run the program
3. Enter value and unit when prompted

## Example
Input: 25 C  
Output:  
Fahrenheit: 77  
Kelvin: 298.15
